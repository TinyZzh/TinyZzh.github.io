Spring Boot Reference Documentation

Spring Boot Version : 2.2.0.RELEASE

Spring Boot Profiles Docs URL: 文档link

1. @Profile 注解

Spring Framework 从3.1版本开始提供@Profile注解。通过可以修饰 @Component、@Configuration和@ConfigurationProperties 三种类型的注解，用于策略的控制程序启动核加载的配置或组件。

4. 总结

  1. 当父类或接口中的定义的方法实现有冲突时, 子类必须覆写父中的方法. 或者把类定义为abstract
  2. 当子类继承父类, 实现父接口的情况时。隐式的继承父类中的方法.
  3. 类中定义的方法实现，优先级高于接口中default实现
